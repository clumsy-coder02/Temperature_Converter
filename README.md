# Temperature Converter

A simple interactive Python application that converts temperature between Fahrenheit and Celsius.

## Features

- Convert temperature from Fahrenheit to Celsius
- Convert temperature from Celsius to Fahrenheit
- User-friendly command-line interface
- Accurate calculations with results displayed to 2 decimal places

## Requirements

- Python 3.x

## Usage

Run the script:
```bash
python main.py
```

Follow the prompts:

1. Enter the temperature reading as a number
2. Enter the unit of measurement:
   - `f` or `F` for Fahrenheit
   - `c` or `C` for Celsius

The program will display the converted temperature.

## Example

```bash
Interactive Temperature Converter

Enter temperature readings: 32
Enter unit for temperature readings 'f' for fahrenheit and 'c' for celsius
Enter unit for temperature readings: f

Current readings: 32.00F
Temperature in degrees celsius: 0.00C
```

## Conversion Formulas

- **Fahrenheit to Celsius:** C = (F - 32) × 5/9
- **Celsius to Fahrenheit:** F = (C × 9/5) + 32
